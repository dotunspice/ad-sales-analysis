**Ad Sales Analysis Project**
**Overview**

This project analyzes the relationship between advertising spend and sales performance across different marketing channels (TV, Radio, and Newspaper).

The goal is to identify which advertising channel has the strongest impact on sales.

**Tools Used**
Python
Pandas
Matplotlib

**Key Insights**
TV advertising has the strongest impact on sales
Radio advertising shows moderate influence
Newspaper advertising has minimal impact

**Conclusion**
TV advertising is the most effective channel for driving sales, followed by radio. Newspaper advertising contributes the least to sales performance.

**Purpose**
This project demonstrates basic data analysis skills, including data exploration, correlation analysis, and visualization using Python.
